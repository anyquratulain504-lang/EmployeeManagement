# Employee Management System

## Overview

This project is a custom Employee Management System developed in Microsoft Dynamics 365 Business Central using AL Language.

The system allows users to manage employee records, validate salary information, and analyze employee data through Business Central Analysis Mode.

## Features

* Employee Table
* Employee List Page
* Employee Card Page
* Employee Salary Field
* Salary Validation
* Employee Record Management
* Analysis Mode Support

## Technologies Used

* Microsoft Dynamics 365 Business Central
* AL Language
* Visual Studio Code
* Docker Container
* Git
* GitHub
